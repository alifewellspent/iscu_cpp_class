# 서울사이버대학교 C++ 프로그래밍 수업

이 저장소는 **서울사이버대학교 C++ 프로그래밍 수업**의 실습 코드를 제공하기 위한 저장소입니다.

수업이 진행되면서 주차별 실습 코드와 안내 문서가 계속 업데이트될 예정입니다.

## 실습 환경

실습 환경을 통일하기 위해 **GitHub Codespaces**를 사용합니다.

Codespaces를 사용하면 별도의 C++ 개발 환경을 각자 컴퓨터에 설치하지 않아도, 브라우저에서 바로 실습 코드를 실행할 수 있습니다.

## Codespace 실행 방법

저장소 화면에서 다음 순서대로 클릭합니다.

**Code → Codespaces → Create codespace on main**

<img width="1801" height="873" alt="GitHub Codespaces 실행 방법" src="https://github.com/user-attachments/assets/e1f91288-c691-4a11-b27b-e096581acffc" />

Codespace가 열리면 아래 명령어를 터미널에 입력합니다.

```bash
cd week01
make run
```

## 실습 종료 후 주의사항

실습이 끝나면 브라우저 탭만 닫지 말고, 반드시 실행 중인 Codespace를 중지해야 합니다.

브라우저 탭만 닫으면 Codespace가 일정 시간 동안 계속 실행될 수 있으며, 이 경우 무료 사용 시간이 소모될 수 있습니다.

### Codespace 중지 방법

1. https://github.com/codespaces 로 이동합니다.
2. 실행 중인 Codespace 오른쪽의 `...` 버튼을 클릭합니다.
3. **Stop codespace**를 선택합니다.
